## RuStore Defold plugin for payments via third-party applications

### [🔗 Developer documentation][10]

### Cloning the repository

Before cloning the repository, download and install the tools:

- [Git](https://git-scm.com/).
- [Git LFS](https://git-lfs.com/).

After installation, run in the command line:

```bash
git lfs install
```

To clone the repository, use the following commands:

```bash
git clone https://gitflic.ru/project/rustore/rustore-defold-pay.git
cd rustore-defold-pay
git lfs pull
```
